HTML Image tag

Html img tag is used to display image on the web page.

HTML Image Example


1.src
It's a neccessary attribute that describe the source or path of the image.
It instructs the browser where to look for the image on the server.

2.alt
the alt attribute defines an alternate text for the image, if it can't be displayed.
The value of the alt attribute describe the image in words.

3.width
It is an optional attribute which is used to specify the width to display the image.
You should apply CSS in place of width attribute.

4.height
It h3 the height of the image. The HTML height attribute also supports iframe, image and object elements.
You should apply CSS in place of height attribute.
