Explanation of formatting element

b element is a physical tag which display text in bold font, without any logical importance, Written in bold tag

Strong tag is a logical tag, which display content in bold font & informs the browser about its logical importance,and this is normal content written outside strong tag.

Write your paragraph in italic text, using i tag

em tag is a logical element,which display the enclosed content in italic font, with added semantics importance.

If you want to mark or highlight a text, you should write content within mark Mark This is mark

anything written within strike element is display with strikethrough It is a thin line which cross the statement.

Monospace font If you want that each letter has same width then you should write content withing tt element.

Superscript text If you put content withing sup element, is shown in Superscript;means it is display half a character's height above the other character.

Subscript text put content within sub element, is shown in subscript; means it is displayed half a character's height below the other characters.

Deleted text Anything that puts within del is display as deleted text.

Inserted text Delete your paragraph. Write another paragraph

Larger text If you want to put your font size not bold but larger than the rest of text then put the content within big. it increase one font size larger than the previous one

Smaller text Write the paragraph in smaller text.
