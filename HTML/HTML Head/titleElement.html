HTML title Element

The HTML title element is used to define the title of the document.
It is used in all HTML/XHTML document.
The title element must be placed between head element, and one document can only have one title element.


What does title element do?

  1. It defines a title in the browser tab.
  2. It provides a title for page when it is added to favorites.
  3. It displays a title for the page in search engine results.

Note: The title element must be specific about the document and its recommended length is 65 to 70 character including spaces.
