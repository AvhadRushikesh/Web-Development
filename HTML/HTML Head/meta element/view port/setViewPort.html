Use meta tag to set the Viewport

This method is introduced to take control over the viewport by using meta tag.
viewport is the user's visible area of a webpage.
It changes from device to device and appers smaller on mobile phones than computer screens.

Syntax for meta viewport element:

meta name="viewport" content="width=device-width, initial-scale=1.0"

Here, the meta viewport element specifies how to control the page's dimensions and scaling.
The width=device-width is used to set the width of the page to follow the screen-width of the device
(Which will vary depending on the device).
The initial-scale=1.0 is used to set initial zoom level when the page is first loaded by the browser.
