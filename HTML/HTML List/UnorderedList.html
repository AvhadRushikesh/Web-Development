HTML Unordered List

HTML Unordered List or Bulleted list display elements in bulleted format.
The HTML ul tag is used for the unoredred list.
There can be 4 types of bulleted list:

ul type = disc

  • disc
  • circle
  • square
  • none

ul type = circle

  • disc
  • circle
  • square
  • none

ul type = square

  • disc
  • circle
  • square
  • none

ul type = none

  • disc
  • circle
  • square
  • none
Note:
-The type attribute is not supported in HTML5, instead of type you can use CSS property of list-style-type.
