Three ways to apply CSS

To use CSS with HTML document, there are three ways:

Inline CSS:
-Defines CSS properties using style attribute in HTML elements.

Internal or Embeded CSS:
-Defines CSS using style tag in head section.

External CSS:
-Defines all CSS property in a seperate .css file, and then include the file with HTML file using tag in section.
