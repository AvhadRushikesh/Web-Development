JavaScript History Object

The JavaScript history object represents an array of URLs visited by the user.
By using this object, you can load previous, forword or any particular page.
The history object is the window property, so it can be access by:
window.history
OR,
history

Property of JavaScript History Object

There are only 1 property of history object.

length
- returns the length of the history URLs.

Methods of JavaScript history object

There are only 3 methods of history object.

  1. forward() - loads the next page.
  2. back() - loads the previous page.
  3. go() - loads the given page number.
