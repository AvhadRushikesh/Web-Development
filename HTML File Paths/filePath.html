HTML File Paths

An HTML file path is used to describe the location of a file in a website folder.
File paths are like an address of file for a web browser.
We can link any external resource to add in our HTML file with the help of file paths such as images, file, CSS file, JS file, video, etc.
The src of href attribute requires an attribute to link any external source to HTML file.

Following are the different types to specify the paths:

img src="picture.jpg"
-It specifies that picture.jpg is located in the same folder as the current page.
img src="images/picture.jpg"
-It specifies that picture.jpg is located in the images folder in the current folder.
img src="/images/picture.jpg"
-It specifies that picture.jpg is located in the images folder at the current root of the current web.
img src="../picture.jpg"
-It specifies that picture.jpg is located in the folder one level up from the current folder.

File paths are used on webpages to link external files like:

  • Web pages
  • Images
  • Style sheets
  • JavaScript

There are two types of file paths:

  1. Absolute File Paths
  2. Relative File Paths
